V3 opens the platform commercially — external tenants, quotas, billing, public sign-up — on top of the V1/V2 pipeline.

## V3 ships

- External tenant signup.
- Public API keys and customer-facing key management.
- Quota enforcement for submissions, storage, and CDN delivery.
- Billing and monetization.
- Public API documentation.
- Public hosted agent streams.
- Legal review and customer-facing license policy.

V3 should reuse V1 tenancy, API-key, approval, and media concepts where they have proven durable.
