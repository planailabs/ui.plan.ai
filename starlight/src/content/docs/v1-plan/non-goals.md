These are not V1 deliverables:

1. Public tenant signup.
2. External customer API keys.
3. Quota enforcement for paid customers.
4. Billing, invoices, pricing pages, or monetization.
5. Server-side frame generation.
6. Server-side metadata generation.
7. Public legal review workflow.
8. Public marketing docs for third-party developers.
9. SDKs beyond curl examples.
10. Webhooks for agent callbacks.

V1 still includes a real internal API. "No public API" no longer describes the product. V1 has a private Agent API for our own agents.
